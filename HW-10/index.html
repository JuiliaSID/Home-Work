Типы селекторов

Селектор определяет, к какому элементу применять то или иное CSS-правило.

Селекторы по типу элемента

Этот базовый селектор выбирает тип элементов, к которым будет применяться правило.

Синтаксис: элемент

Пример: селектор span выберет все элементы c тегом "span"

Селекторы по идентификатору

Этот базовый селектор выбирает элементы, основываясь на значении их id атрибута. Не забывайте, что идентификатор должен быть уникальным, т. е. использоваться только для одного элемента в HTML-документе.

Синтаксис: #имяИдентификатора

Пример: селектор #name выберет элемент с идентификатором name (который был определён в атрибуте id="name").

Селекторы по классу

Этот базовый селектор выбирает элементы, основываясь на значении их атрибута class.

Синтаксис: .имяКласса

Пример: селектор .index выберет все элементы с соответствующим классом (который был определён в атрибуте class="index").

Селекторы групп

Чтобы применить одинаковый стиль сразу к нескольким элементам веб-страницы используется групповой селектор. Конечно, в перечислении могут участвовать не только селекторы тегов, но и классы, и идентификаторы. Если нужно стилизовать абсолютно все элементы страницы, можно задействовать универсальный селектор *.

Синтаксис: элемент, елемент / .имяКласса, .имяКласса

Пример: color-first, color-second выберет оба элемента для стилизации текста

Селекторы групп

Комбинатор ' ' (пробел) выбирает элементы, которые находятся внутри указанного элемента (вне зависимости от уровня вложенности).

Синтаксис: A B

Пример: селектор div h5 выберет все элементы "h5", которые находятся внутри элемента "div".
